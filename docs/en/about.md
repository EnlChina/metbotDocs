# About

MetBot is written in JavaScript, using the [grammy bot framework](https://grammy.dev/)

The miniapp uses [Vue](https://vuejs.org/) + [daisyUI](https://daisyui.com/)

This documentation is generated by [rspress](https://rspress.dev/)

The bot source code is on [GitHub](https://github.com/EnlChina/AgentMetBot). You need to join the [EnlChina](https://github.com/EnlChina) organization to access it. Contributions are welcome!

The [bot](https://t.me/AgentMetBot), miniapp and bot documentation are all deployed on Cloudflare. Thanks to Cloudflare!
